# Running the Visual Agent System

## Setup (First Time Only)

1. Install LLM API dependencies:
   cd /home/rv/projects/vision-systems/group-project
   python3.12 -m venv .venv-llm
   source .venv-llm/bin/activate
   pip install -r requirements-llms.txt

2. Install Backend dependencies:
   cd web_system
   python3 -m venv .venv-web-system
   source .venv-web-system/bin/activate
   pip install -r backend/requirements.txt

3. Install Frontend dependencies:
   cd frontend
   nvm use 20  # or ensure Node.js 20+ is active
   npm install

## Running the System

Terminal 1 - LLM API (Port 5000):
   cd /home/rv/projects/vision-systems/group-project
   source .venv-llm/bin/activate
   python vlm_api.py

Terminal 2 - Backend (Port 8080):
   cd /home/rv/projects/vision-systems/group-project/web_system/backend
   source ../.venv-web-system/bin/activate
   python -m uvicorn app.main:app --host 0.0.0.0 --port 8080 --reload

Terminal 3 - Frontend (Port 5173):
   cd /home/rv/projects/vision-systems/group-project/web_system/frontend
   nvm use 20
   npm run dev

## Access
Open browser: http://localhost:5173

## Stop
Press Ctrl+C in each terminal
